# SaaS Vision

## Doel
weFactly als SaaS platform:
- multi-tenant
- centraal beheer
- schaalbaar

## Functionaliteiten
- meerdere bedrijven
- gebruikersbeheer
- rollen en rechten
- centrale opslag
- automatische updates

## Mogelijke stack
Appwrite kan gebruikt worden voor:
- auth
- user management
- sessions
- storage

## Belangrijk
- core blijft eigen systeem
- SaaS is een laag erbovenop

## Abonnementen (concept)
### Free
- beperkte functies
- beperkt gebruik

### Basic
- volledige core functies

### Pro
- modules
- uitbreidingen

### Business
- meerdere users
- advanced features
