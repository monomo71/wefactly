# Scope – Sellable v1

## Doel
Gebruiker moet:
- snel een factuur kunnen maken
- deze professioneel kunnen versturen
- overzicht hebben

## Functionaliteiten
### Klanten
- lijst
- toevoegen
- bewerken

### Producten
- lijst
- toevoegen
- prijsinformatie

### Offertes
- maken
- opslaan
- status

### Facturen
- maken
- opslaan
- status (draft / sent / paid / overdue)

### PDF generatie
- nette documenten
- professioneel uiterlijk

### Mail
- versturen van offertes en facturen

### Dashboard (basis)
- info tegels
- snelle acties
- recente items

## Belangrijk
Dit moet:
- bruikbaar zijn
- compleet aanvoelen
- verkoopbaar zijn

Maar:
- niet alles bevatten
