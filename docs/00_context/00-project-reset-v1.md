# 003_weFactly – Project Reset v1

## Wat is weFactly
weFactly is een facturatieplatform met als kern:
- klanten
- producten
- offertes
- facturen

Het doel is om ondernemers zonder boekhoudkennis snel en professioneel te laten werken in een systeem dat logisch voelt en prettig in gebruik is.

## Belangrijke eigenschappen
- sterke focus op UX
- overzichtelijk en intuïtief
- snel en responsief
- modulair uitbreidbaar
- professionele uitstraling

### v3

## Les uit eerdere versies
- scope moet strak blijven
- UX moet vroeg vaststaan
- structuur is verplicht
- complexiteit moet gefaseerd worden

## Doel van deze reset
Een stabiele basis creëren die:
- bruikbaar is
- verkoopbaar is
- uitbreidbaar is
- niet opnieuw vastloopt
