# Modular Vision

weFactly is ontworpen om uitbreidbaar te zijn via modules.

## Wat modules mogen
- extra velden toevoegen
- extra UI-secties toevoegen
- logica uitbreiden
- instellingen uitbreiden

## Wat modules NIET mogen
- core UX breken
- navigatie veranderen
- inconsistent gedrag introduceren

## Extension points
Core onderdelen krijgen vaste uitbreidpunten:
- detailpagina’s
- formulieren
- instellingen
- acties

## Voorbeelden
### Voorraadmodule
Voegt toe aan producten:
- voorraad
- inkoopprijs
- leveranciersrelaties

### Webshopmodule
Voegt toe aan producten:
- EAN
- afbeeldingen
- varianten
- koppelingen

## Belangrijk principe
> Open voor uitbreiding, streng in structuur
