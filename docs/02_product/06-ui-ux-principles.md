# UI / UX Principles

## Doel
weFactly moet:
- prettig werken
- duidelijk zijn
- professioneel ogen
- vertrouwen geven

## Basisregels
### Consistentie
- Alle tabellen hetzelfde gedrag
- Alle formulieren dezelfde structuur
- Acties altijd op dezelfde plek

### Duidelijkheid
- Elke knop is logisch
- Elke actie is voorspelbaar
- Geen verborgen gedrag

### Tooltips
- Alle iconen krijgen uitleg
- Korte, duidelijke teksten

### Snelheid
- Directe feedback
- Geen vertraging
- Applicatiegevoel

### Minimalisme
- Geen overload
- Rustige layouts
- Focus op taak

## Visuele stijl
- Strak (niet te rond)
- Rustige kleuren
- Subtiele animaties
- Professioneel

## UX regel
> Als iets uitleg nodig heeft, is het niet goed genoeg ontworpen.
