# Setup Prompt

Gebruik deze prompt als eerste echte bouwprompt in Copilot:

Bouw de fundering van weFactly als React + Vite applicatie met een duidelijke, schaalbare structuur.

Doelen:
- Single-tenant first
- SaaS-aware
- Modulair voorbereid
- Strakke professionele UI basis
- Tailwind + aanpasbare componentlaag
- Geen business features buiten fundament

Maak:
1. Projectstructuur voor app, components, modules, pages, layout, hooks, lib, api, styles
2. Sidebar layout met header
3. Basis routing
4. Placeholder pagina’s voor:
   - Dashboard
   - Klanten
   - Producten
   - Offertes
   - Facturen
   - Instellingen
5. Design tokens / thema basis
6. Tabel- en formulierbasiscomponenten
7. Tooltip support
8. Environment config basis
9. Auth placeholder laag, nog simpel maar voorbereid op later
10. Heldere mappenstructuur en comments waar uitbreiding verwacht wordt

Belangrijk:
- Houd de code netjes en uitbreidbaar
- Niet te rond vormgeven
- Rustige zakelijke uitstraling
- Geen mock chaos
- Geen overengineered patterns
- Geef per stap kort aan waarom de structuur zo is gekozen
