# Copilot Start Prompt

Gebruik deze prompt in VSCode Copilot:

Je werkt aan weFactly, een modulair facturatieplatform.

BELANGRIJK:
- Bouw alleen wat in de huidige fase van de build-order hoort
- Houd code simpel en uitbreidbaar
- UX is prioriteit
- Geen overengineering
- Gebruik duidelijke structuur
- Respecteer projectdocumenten

Regels:
- Duidelijkheid > flexibiliteit
- Geen complexe features buiten scope
- Geen hardcoded data
- Geen directe database calls vanuit UI
- Houd rekening met toekomstige SaaS en modules

Start met:
Fase 1 — Fundament
- setup project
- layout
- basis structuur
- design system

Werk stap voor stap.
Stop na elke fase en evalueer.
