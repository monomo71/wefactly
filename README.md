# 003_weFactly

Structured project foundation for weFactly.

## Structure

- `00_context/` — reset and project context
- `01_foundation/` — product philosophy, scope, exclusions
- `02_product/` — modular vision and UI/UX principles
- `03_architecture/` — technical direction, data model, database, security
- `04_execution/` — build order, copilot prompts, setup
- `05_growth/` — roadmap, SaaS vision, marketplace, open-source/distribution
- `99_reference/` — GitHub workflow, wishlist

## Current goal

Build a sellable v1 that feels professional, is self-hostable, and is designed to grow toward SaaS and a marketplace without rewrite pressure.
